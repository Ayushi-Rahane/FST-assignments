Google REST APIs Explorer

Interact programmatically with Google Apps — Drive, Calendar, Slides, Sheets & Docs — using RESTful web services.

Step 1 — Authenticate with Google

Sign in with your Google account using OAuth 2.0 to get an access token for calling Google APIs.

Step 2 — Explore Google REST APIs

Google Drive

List files from your Drive

GETgoogleapis.com/drive/v3/files

Sends HTTP GET to Drive API v3. Returns JSON list of files (name, id, mimeType).

Google Calendar

Get primary calendar metadata

GETgoogleapis.com/calendar/v3/calendars/primary

Sends HTTP GET to Calendar API v3. Returns metadata (summary, timeZone, id).

Google Slides

Create a blank presentation

POSTslides.googleapis.com/v1/presentations

Sends HTTP POST to Slides API v1. Creates a blank presentation. Verify in Drive.

Google Sheets

Create a new spreadsheet

POSTsheets.googleapis.com/v4/spreadsheets

Sends HTTP POST to Sheets API v4. Creates a spreadsheet. Open returned URL to verify.

Google Docs

Create a blank document

POSTdocs.googleapis.com/v1/documents

Sends HTTP POST to Docs API v1. Creates a blank Google Doc. Verify in Drive.
